Cara Install Tanpa Tampilan URL

Web app ini sudah mendukung fitur PWA (Progressive Web App). Setelah di-upload ke server hosting, Anda bisa membukanya seperti aplikasi HP/Desktop bawaan:

  1. Di Android (Chrome): Klik ikon titik tiga di kanan atas Chrome, pilih "Tambahkan ke Layar Utama" atau "Instal Aplikasi".
  2. Di Laptop (Chrome/Edge): Klik ikon komputer kecil dengan tanda panah ke bawah yang muncul di sisi kanan kolom alamat URL browser Anda, lalu pilih "Instal".
  3. Aplikasi akan terpasang di menu utama Android / Desktop Anda. Saat dibuka, aplikasi akan berjalan layar penuh (Standalone) tanpa bar URL browser sama sekali!
